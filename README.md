## Gymbro (Expo React Native)

Run on iPhone with Expo Go:

1. Install Node.js LTS from `https://nodejs.org/`
2. Install dependencies:
   - In this folder, run: `npm install`
3. Start Expo (Tunnel works best for iPhone):
   - `npx expo start --tunnel`
4. On your iPhone, install Expo Go from the App Store, open it, and scan the QR code.

Notes:
- If asked for camera or library permissions, allow them.
- The app has three tabs: Feed, Take Photo, Profile.
- Posting a photo will add it to the feed and your profile.
- Tap a user's name or avatar in the feed to open their profile.
